# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x.x | ✅ |
| <1.0 | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

Email:
aynalhaque7411@gmail.com

Please include:

- Description of the issue
- Steps to reproduce
- Affected version
- Screenshots (if available)

We will acknowledge your report within 72 hours and work to resolve valid issues as quickly as possible.

Please do not disclose the vulnerability publicly until a fix has been released.
